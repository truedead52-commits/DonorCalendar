# 🩸 Календарь донора

Android-приложение для учёта донаций крови с автоматическим расчётом доступных дат.

## Как открыть и запустить

### Требования
- Android Studio Hedgehog (2023.1.1) или новее — [скачать бесплатно](https://developer.android.com/studio)
- Android SDK (устанавливается вместе с Android Studio)

### Шаги

1. **Распакуйте архив** `DonorCalendar.zip` в удобную папку.

2. **Откройте в Android Studio:**
   - File → Open → выберите папку `DonorCalendar`
   - Подождите окончания синхронизации Gradle (первый раз скачивает зависимости — 2–5 минут)

3. **Запустите приложение:**
   - На реальном телефоне: подключите через USB, включите «Отладку по USB» в настройках разработчика, нажмите ▶ Run
   - На эмуляторе: Device Manager → создайте устройство → ▶ Run

### Сборка APK для установки без Android Studio

В терминале в папке проекта:
```bash
# Mac / Linux
./gradlew assembleDebug

# Windows
gradlew.bat assembleDebug
```
APK будет в `app/build/outputs/apk/debug/app-debug.apk`

## Функциональность

| Экран | Что умеет |
|-------|-----------|
| **Статус** | Показывает доступность каждого типа донации. Зелёный — можно сдавать сейчас, красный — через N дней с датой |
| **История** | Список всех донаций. Удаление по свайпу влево или кнопкой 🗑️ |
| **Добавить** | Кнопка FAB — выбор даты и типа донации, сохранение в базу данных |

## Правила расчёта интервалов

| После \ До | Кровь | Плазма | Тромбоциты |
|------------|-------|--------|------------|
| **Кровь**      | 60 дн | 30 дн  | 30 дн      |
| **Плазма**     | 14 дн | 14 дн  | 14 дн      |
| **Тромбоциты** | 14 дн | 14 дн  | 14 дн      |

Лимит цельной крови: **5 раз/год** (мужчины), **4 раза/год** (женщины).

## Архитектура

```
data/       — Room БД, DataStore, Entity, DAO, Enum-типы
domain/     — Бизнес-логика расчётов (DonationRules)
ui/         — Jetpack Compose экраны + ViewModel
```

Стек: **Kotlin · Jetpack Compose · Material 3 · Room · DataStore · MVVM · StateFlow**
